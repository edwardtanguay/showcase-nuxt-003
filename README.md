## Setup

```bash
npm i
npm run dev
```

## Features

- Nuxt 4 with Nuxt UI
- TypeScript
- Tailwind CSS
- dark/light mode (useColorMode)
- responsive navigation
- interactive API forms with error handling
- one-click publishing to Vercel

## Change layout colors

- see `app.config.ts`

## Deployment

- simply push to Vercel, all defaults will work
